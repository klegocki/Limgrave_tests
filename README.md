This project is a text-based RPG heavily inspired by the starting area of Elden Ring. While the game features a fully playable combat system, its primary purpose was to serve as a comprehensive environment for mastering Unit Testing using pytest.

Core Game Mechanics:

-Diverse Combat: Systems for physical (swords) and magical (spells) attacks.

-Dynamic Enemy AI: Various opponents with unique stats and behaviors.

-RPG Systems: Loot drop mechanics and an inventory system for consumable items (potions).

Technical Focus (Testing):

-Built with Python and a robust testing suite using Pytest.

-Achieved high code coverage by testing edge cases in combat logic, RNG-based drop rates, and player state management.
